[Tube Archivist](https://github.com/tubearchivist/tubearchivist)

Requires API key.

Allowed fields: `["downloads", "videos", "channels", "playlists"]`.

```yaml
widget:
  type: tubearchivist
  url: http://tubearchivist.host.or.ip
  key: apikeyapikeyapikeyapikeyapikey
```
